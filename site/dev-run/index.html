    Запуск для разработки

    Обычный режим

    ./scripts/launch.sh dev

    Фоновый режим

    ./scripts/launch.sh dev --detach

    Скрипт запускает:

    • мастер‑бота;
    • REST API сервер;
    • frontend‑приложение.
