# Инструкция по использованию

## Шаг 1. Подключение основного бота

- Найдите развернутого основного бота GraceHub Platform в Telegram.  
- Нажмите **Start** или отправьте `/start`.  
- Бот создаст личный кабинет и даст базовые инструкции.

## Шаг 2. Регистрация бота поддержки

- В основном боте выберите опцию добавления нового бота.  
- Получите токен через @BotFather.  
- Отправьте токен боту GraceHub Platform для регистрации.

## Шаг 3. Инициализация администратора

- Напишите `/start` вашему боту поддержки.  
- Ваш аккаунт будет отмечен как администратор.

## Шаг 4. Создание суперчата с топиками

- Создайте новую группу в Telegram.  
- Включите режим обсуждений (Topics).  
- Добавьте бота поддержки с правами администратора и правами на управление сообщениями и топиками.

## Шаг 5. Привязка бота к топику General

В топике General отправьте:

/bind @your_bot_username


После привязки обращения клиентов будут попадать в этот суперчат с распределением по топикам.
